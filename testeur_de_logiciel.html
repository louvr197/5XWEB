TESTEUR DE LOGICIEL

Le testeur de logiciel est une personne ayant pour mission de repérer les erreurs dans les logiciels produits par d’autres informaticiens ainsi que l’adhérence au cahier des charges du logiciel.





Un testeur de logiciel va produire un rapport pour chaque version du produit contenant les erreurs logicielles mais aussi des conseils qualité et des démonstration concrètes des cas d’utilisations. En sommes, il va vérifier que le programme fonctionne et réponds bien à toutes les attentes du client.



Afin de réaliser ce rapport, le testeur logiciel va donc identifier, implémenter, installer et exécuter les tests les plus adaptés. Il va ensuite interpréter les résultats de ces tests.


Les tests réalisés peuvent être classés dans plusieurs groupes :

 - Les tests fonctionnels pour vérifier l’adhérence au cahier des charges.
 - Les tests de performance pour stresser le programme avec une grande quantité de travail
 - Les tests de fiabilité pour vérifier des cas rares qui pourraient causer des problèmes dans l’exécution de programme auxquels le créateur n’aurait pas pensé.
 - Les tests linguistiques pour vérifier la qualité de l’internationalisation du programme.

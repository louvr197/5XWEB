TEMPLATE WEB

Un gabarit permet de définir une structure réutilisable pour assurer la cohérence d’un site web. L’apparence du site web étant définie par le gabarit, seul le contenu doit être écrit pour une nouvelle page et un changement du gabarit affecte toutes les pages du site web.




Un « template », ou « gabarit » en français est, dans le contexte informatique, un moyen de séparer le fond de la forme. La présentation des pages web est souvent gérée par des documents CSS (Cascading Style Sheet) remplie par le développeur mais peut aussi être fournie préremplie par les outils de créations de site.

MySQL


Prononcé “M-aïe es-ku-elle" en français et “M-aïe sikwel” à l’internationale, MySQL est la base de données Open-source la plus utilisée. MySQL est développée par Oracle



Une base de données est un ensemble structuré d’informations organisées pour être facilement accessibles, gérées et mises à jour. MySQL utilise un modèle de base de données relationnelle. C’est à dire que les données sont enregistrées dans des tables distinctes contenant des données similaires et leurs relations avec des données dans d’autres tables. Ces relations peuvent lier des entrées selon des critères différents selon les besoins. Le stockage de ces tables dans des fichiers physiques différents plutôt que dans un unique grand fichier permets une optimisation impossible dans d’autres modèles de base de données.


Le SQL de MySQL signifie “structured query language” (langage de requêtes structurées). Le SQL est le langage standard le plus utilisé pour gérer les accès aux bases de données.

MySQL étant open-source, le programme et son code peuvent être obtenu gratuitement et modifié par les utilisateurs sous licence GNU GPL. Une License commerciale est également disponible si elle correspond mieux aux besoins de l’utilisateur.


Cet état de logiciel open-source a fait qu'au cours de son existence MySQL a été rendu compatible avec la plupart des langages informatiques.
